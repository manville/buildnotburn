# **App Name**: BuildNotBurn CFML

## Core Features:

- Add Brick: Add a new 'brick' (task) to the task list. Accepts input via a form and stores it in bricks.json.
- Remove Brick: Allows the user to mark a brick as complete (remove it from view) from bricks.json.
- Display Bricks: Fetches and displays all 'bricks' from bricks.json in a HTML list.

## Style Guidelines:

- Background color: #0f0f0f
- Surface color: #1a1a1a
- Border color: #2d2d2d
- Accent color: #ff6b00 (Safety Orange)
- Headers font: 'Oswald' (sans-serif) in bold and all caps.
- Body font: 'Roboto Mono' (monospace) for a code-style look.
- CSS Grid Pattern background using a linear gradient of 1px on #0f0f0f.
